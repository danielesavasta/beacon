description: "A modular visual system translating the letters of the Latin alphabet into abstract geometric forms. Each glyph is built from a shared vocabulary of shapes — circle, line, arc — that can be recombined to produce new letterforms. The system explores legibility at the edge: how far can a letter be abstracted before it ceases to communicate?"
